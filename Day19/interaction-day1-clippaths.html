Clip-Paths

Getting out of rectangular world
  • Circle
  • Polygon
  • Path
  • Inset

Idea: we limit the displayed area of an element by applying a shape to clip on

  • circle (radius at inset-left inset-top
  • ellipsis (radius-x radius-y at inset-left inset-right)
  • inset( top right bottom left )
  • polygon( list of pair coordinates )
  • path (SVG path syntax)

Clip paths can be applied to block and inline elements
