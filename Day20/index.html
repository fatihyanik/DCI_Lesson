Transforming Elements

Idea

Idea: Use CSS property transform, to change the visual appearance of HTML block level elements.

Ways to transform

  • Rotation. needs an angle (Winkel), and a choice of the axis, we want to rotate around, X and Y need a distance to the viewer (perspective-function)
    transform: perspective(500px) rotateX(45deg)
  • Translations: is "shifting" an element along an axis. (Z again takes a distance with perspective)
    transform: translateX(100px);
  • Scaling: making things "bigger" or "smaller"
    transform: scale(2);
  • Skewing: tilt/shift/collapse a shape with a given angle along an axis
    transform: skewX(45deg);

Things to consider

3D-transformations (everything that includes Z-dimension) needs a perspective()-call with a parameter describing the distance of the 2D surface to the viewer
transform: perspective(15cm) rotateY(45deg);

view-port (screen-width) effects 3D-transformations in combination with perspective-parameter

you can control, per element where the X and Y axis meet. CSS-property transform-origin
transform-origin: 20px 50px;

the order of functions applied to transform property matter.

an element can only have one value for the transform-property. transform: rotateY(45deg) translateX(50px) scale(1.5); transform: rotateY(45deg);
transform: translateX(50px);
transform: scale(1.5);
